 🧠 这是什么？

这是一个个人认知操作系统的开源项目。它基于能量最优的第一性原理，旨在通过一套清晰的协议和框架，帮助个体高效管理人际互动与认知资源。

它不是成功学，而是一套可操作的“人性系统工程”工具。

 ⚙️ 核心架构

- 第一性原理：节能主义
- 决策闭环：Ne（扫描） -> Fi（审判） -> Te（执行）
- 行为循环：高敏感 -> 强边界 -> 快执行 -> 善复盘
